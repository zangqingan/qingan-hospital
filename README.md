# 医院管理系统
### 1-前台
1. 首页
2. 医院介绍
3. 新闻中心
4. 预约挂号
### 2-后台
1. 用户管理，添加删除修改用户资料
2. 权限管理，用户有哪些的权限，有些页面只有某部分特殊用户可以访问。
3. 登陆注册
4. 新闻信息管理
5. 新闻分类管理
6. 医生管理
7. 患者管理
8. 挂号管理
9. 药品管理
10. 财务统计
### 3-权限管理
权限管理需要的数据表
1. 用户表
2. 角色表
3. 权限表
举例：
张三用户是1个管理员，那么管理员就有（编辑用户，编辑新闻，编辑挂号，查看统计数据）
李四用户是1个医生，那么就拥有编辑挂号的权限。
王五是1个新闻编辑者，那么就拥有新闻编辑的权限。
设置权限表
1添加/删除 /需改权限
2字段字段：
权限编码，id
编辑用户权限，权限名称
/admin/users/userlist2，访问路径
